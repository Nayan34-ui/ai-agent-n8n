# ai-agent-n8n
Free AI Agent built with n8n + Gemini + SerpAPI and with simple memory and calculator.
A fully functional AI Agent built for free!

## Tools Used
- Google Gemini 2.0 — AI Brain
- SerpAPI — Real-time Google Search  
- Calculator — Math operations
- Simple Memory — Conversation memory
- n8n — Workflow automation

## How to use
1. Import the JSON file into your n8n instance
2. Add your own API credentials
3. Start chatting!
